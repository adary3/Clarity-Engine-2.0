# Clarity-Engine-2.0
What it is
The Clarity Engine is a free web tool that helps you articulate who you are, what drives you, and what you want to build. You answer 21 honest questions across three sections — identity, vision, and idea. At the end, an AI expert reads your answers and writes a deep personal synthesis of who you actually are.
No login. No data stored. No fluff.

How to use it
Open the tool in any browser. Answer the questions honestly — the more honest, the more useful the report. You need to answer at least 15 questions before the report generates. When you're done, choose your AI analyst, paste your API key, and generate your Clarity Report. Download it as a text file when finished.

API Keys — Expert Synthesis
The expert synthesis requires one API call at the end. Your key is never stored — it lives only in your browser for that session.
Gemini (Free): Get your key at aistudio.google.com → Get API Key. Takes 30 seconds. No credit card.
Claude (Premium): Get your key at console.anthropic.com → API Keys. Add $5 credit. One report costs roughly $0.02.
